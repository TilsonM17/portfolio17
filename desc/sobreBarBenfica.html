Sistema de Gestão do Bar do Benfica.

Não tenho autorização para exibir demais o sistema. Por se tratar de um sistema interno e em funcionamento.

Este Sistema foi construido usando as Tecnologias/Ferramentas:

  • C# .
  • SQLSERVER..
  • Bunifu Framework UI tools..
  • Inno Setup..
  • Draw IO..


Algumas das suas funções para o Modulo Funcionario são:

  • Registro de Produtos(Bebidas/Pratos).
  • Inserção e controle de Funcionarios.
  • Inserção de Vendas.
  • Registro de Saidas de Activos.
  • Emissão de Factura.
  • Registro de Vendas.
  • Separação de Responsabilidades(Admin/Funcionario).

Algumas das suas funções para o Modulo Administrador são:

  • Registro de Faltas do Funcionario.
  • Pagamento de Salarios.
  • Actividades dos Funcionario.
  • Inserção de Vendas.
  • Balanço Semanal/Mensal das Contas.
